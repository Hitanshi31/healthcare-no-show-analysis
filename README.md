# Healthcare Appointment No-Show Analysis

## Project Overview

This project analyzes healthcare appointment data to identify patterns and factors contributing to patient no-shows.

The project focuses on:
- patient attendance behavior
- waiting time analysis
- demographic insights
- operational inefficiencies
- healthcare scheduling trends

The analysis was performed using Python and Power BI.

## Business Problem

Missed medical appointments create operational inefficiencies and revenue loss for healthcare providers.

This project aims to identify:
- why patients miss appointments
- which factors influence attendance
- how healthcare scheduling efficiency can be improved

## Tools & Technologies

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook
- Power BI

## Project Workflow

1. Data Cleaning
2. Feature Engineering
3. Exploratory Data Analysis
4. Insight Generation
5. Power BI Dashboard Development

## Feature Engineering

- WaitingDays
- WaitingCategory
- AppointmentWeekday
- AgeGroup

## Key Insights

- Longer waiting periods were associated with higher no-show rates.
- Patients with chronic conditions showed better appointment attendance.
- Certain neighborhoods demonstrated significantly higher no-show percentages.
- Appointment attendance varied across weekdays and age groups.
